# TEM-Nanoparticles-Classification
ResNet50-based classification of TEM nanoparticle images (Diamante vs MWCNT). Expanded dataset using heavy Albumentations augmentation (200 images/class) with 75:15:15 split. Trained with AdamW, LR scheduling &amp; early stopping. Achieved 98.33% test accuracy (F1≈0.983). Includes Grad-CAM for visual interpretability.
The paper can be found using the DOI: 10.21203/rs.3.rs-7437155/v1 (https://doi.org/10.21203/rs.3.rs-7437155/v1)
